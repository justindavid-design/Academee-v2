# Quiz Builder User Guide

## Getting Started

### Step 1: Access the Quiz Builder
1. Go to your Course Dashboard
2. In the Stream section, locate the **Create Content** area
3. Click the **Quiz** button

### Step 2: Choose Your Question Format
You'll see a beautiful selection screen with four question types:

- **Multiple Choice** - One correct answer
- **Checkbox** - Multiple correct answers  
- **True/False** - Binary yes/no questions
- **Short Answer** - Open-ended text responses

Click on your desired question type to proceed.

## Building Your Quiz

### The Quiz Builder Interface

#### Top Header
- **Quiz Title Field** - Enter your quiz name (auto-saves)
- **Auto-save Indicator** - Shows save status in real-time
- **Save Draft** - Manually save your progress
- **Publish** - Finalize and make quiz available to students

#### Left Section (Main Content)
- **Quiz Settings** - Configure time limits, due dates, passing score
- **Questions** - Your quiz questions displayed as cards

#### Right Sidebar
- **Quiz Summary** - View total questions and points
- **Question Navigator** - Jump to any question
- **Quick Stats** - See averages and counts

## Adding Questions

### Creating Your First Question
1. Fill in the question text
2. Enter the number of points
3. Add answer choices
4. Select the correct answer(s)
5. Click the arrow to collapse the question

### Adding More Questions
1. Click **"+ Add Question"** at the bottom
2. Select the question type (or keep the current type)
3. Fill in the question details
4. Repeat as needed

### Managing Questions

#### Moving Questions
- Click and drag the grip handle (three horizontal lines)
- Drag up or down to reorder
- Drop in the new position

#### Editing Questions
- Click on any question card to expand it
- Edit text, points, or answers
- Changes auto-save every 5 seconds

#### Deleting Questions
- Expand a question
- Scroll to the bottom
- Click **Delete Question**
- Confirm deletion

## Question Types Explained

### Multiple Choice
**When to use:** Straightforward factual questions with one clear answer

**How it works:**
1. Students see one question with multiple options
2. They select exactly one answer
3. Only one option should be marked as correct

**Example:**
> What is the capital of France?
> - London
> - Berlin
> - Paris (correct)
> - Madrid

### Checkbox
**When to use:** Questions where multiple answers can be correct

**How it works:**
1. Students see multiple options
2. They can select as many as apply
3. Mark all correct answers as correct
4. All must be selected for full credit

**Example:**
> Which of these are programming languages? (Select all that apply)
> - Python (correct)
> - JavaScript (correct)
> - Photoshop
> - Java (correct)

### True/False
**When to use:** Simple yes/no or true/false statements

**How it works:**
1. Students see a statement
2. They choose True or False
3. Select which answer is correct
4. Quick assessment option

**Example:**
> The Earth is flat.
> - True
> - False (correct)

### Short Answer
**When to use:** Open-ended questions requiring typed responses

**How it works:**
1. Students type their response
2. Instructor reviews and grades manually
3. Note: This doesn't auto-grade (for now)
4. Great for essays, explanations, critical thinking

**Example:**
> Explain photosynthesis in 2-3 sentences.
> [Expected answer: Students should mention light energy, chlorophyll, glucose production, oxygen release]

## Configuring Quiz Settings

### Basic Settings
- **Description** - Optional brief explanation for students
- **Time Limit** - Minutes allowed (leave blank for unlimited)
- **Due Date** - When the quiz closes
- **Attempts Allowed** - How many times students can retake
- **Passing Score** - Percentage needed to pass

### Points & Scoring
- Set points per question (default: 1)
- Total points displays in sidebar
- Passing score is calculated as percentage

**Example:**
> 5 questions × 2 points each = 10 total points
> Passing score 70% = need 7 points to pass

## Saving Your Work

### Auto-Save
- Quiz auto-saves every 5 seconds
- Look for the green "Saved" indicator in the header
- Your progress is never lost

### Save Draft
- Click **Save Draft** to manually save
- Quiz won't be visible to students yet
- Can edit later

### Publish
- Click **Publish** when ready
- Quiz becomes available to students immediately
- Cannot unpublish (but can delete questions)

## Tips & Best Practices

### Question Design
✓ Be clear and specific in wording
✓ Avoid double negatives
✓ Keep answer options similar in length
✓ Use consistent formatting
✓ Test your questions before publishing

### Difficulty Balance
✓ Mix easy, medium, and hard questions
✓ Start with easier questions
✓ Build toward more challenging content
✓ Consider your audience's level

### Timing
✓ Calculate realistic time per question
✓ Add buffer time for reading
✓ Test the time limit yourself
✓ 1-2 minutes per question is typical

### Grading
✓ Set realistic passing scores
✓ Consider question difficulty
✓ For short answer, grade consistently
✓ Provide feedback to students

## Troubleshooting

### Quiz Not Saving
- Check your internet connection
- Try clicking "Save Draft" manually
- Refresh the page and try again
- Check browser console (F12) for errors

### Lost My Work
- All changes auto-save to our servers
- Refresh the page to reload
- Contact support if still missing

### Can't Publish
- Ensure quiz title is filled in
- Make sure you have at least one question
- All questions need text
- All questions need correct answer(s) selected

### Answers Getting Mixed Up
- Reorder questions using drag-and-drop
- Questions always display in the order you set
- Student view will match builder view

## Creating Different Quiz Types

### Quick Check (5 minutes)
- 5-10 True/False questions
- 1 point each
- For daily concept check

### Mini Quiz (15 minutes)
- Mix of Multiple Choice and Short Answer
- 10 questions
- 2-3 points each
- For weekly review

### Unit Exam (1 hour)
- Mix of all types
- 30-40 questions
- Various points (1-5)
- Comprehensive coverage

### Exit Ticket (3 minutes)
- 1-2 short answer questions
- Check understanding before leaving
- Optional grading
